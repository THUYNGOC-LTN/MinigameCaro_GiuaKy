# Minigame Caro (5-in-a-row)

## Cấu trúc
- server.py
- client.py
- game_logic.py

## Chạy thử
1. Mở terminal và chạy server:
   ```bash
   python server.py
   ```
2. Mở 2 terminal khác để chạy 2 client:
   ```bash
   python client.py
   ```
3. Nhập tọa độ: `row col` (0–14)

## Mục tiêu
Xếp 5 quân liên tiếp ngang, dọc hoặc chéo để thắng.
